[TOOL USAGE RULES]
1. Use the `search_memory` tool when you need more context about past interactions, user preferences, or specific knowledge stored in the memory vault.
2. Use `rag_search` to perform a hybrid vector and text search to retrieve factual context from uploaded documents.
3. This ensures your responses are accurate, personalized, and context-aware.
4. You can provide a conversational response while or after checking memory; do not feel strictly obligated to check memory before every single message if it's not necessary (e.g., for simple greetings).
